## Themes

### Stout

`Stout` is a dark color theme inspired by one of the dark beer style Stout.

![Stout](Stout.png)

## Usage
```
$ git clone https://github.com/86/Xcode-themes.git
$ cd xcode-themes
$ cp *.dvtcolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
```
